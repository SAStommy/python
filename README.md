# Week 3

+ **list(變數['a','b'.'c'])**
  + 變數[start:end] : 取從start剄end
  + 變數[:n] : 取前n名
  + 變數[:-n] : 取前n名,不取後n項
  + 變數[n:] : 取n項到最後
  + 變數[-n:] : 不取前n名,取後n項
  + 變數[:] : 取所有元素
  + 變數[start : end : step] : 每隔step從start到end
